# Roadmap

## Current Version: v1.0 (Q1 2026)

### Completed Features

#### Desktop App (7 Views)
- [x] Project Hub (capability tree, entity connections)
- [x] Task Manager (list, kanban, timeline, detail views)
- [x] Knowledge Browser with tree navigation
- [x] Ticket Queue (status workflow, convert to tasks)
- [x] Training Room (incidents, lessons, skills, rules)
- [x] Settings
- [x] Worktree Status

#### MCP Server (20 Tools)
- [x] Knowledge tools (context, search, list, get, modules)
- [x] Task tools (create, update, list, get, complete)
- [x] Impact analysis (analyze, check_gate, list)
- [x] Ticket tools (create, list, update, convert_to_task)
- [x] Training tools (lesson_create, rule_check)
- [x] Session launch

#### CLI
- [x] `sidstack init` with governance templates
- [x] `sidstack init --scan` AI-powered knowledge generation
- [x] Governance commands (show, check)
- [x] Knowledge commands (list, create, templates)

## Design Principles

1. **Local-First** - No cloud dependencies
2. **Knowledge-Driven** - Understand before automate
3. **MCP-Native** - Claude Code integration via MCP tools
4. **Governance-Forward** - Quality gates enforced automatically

---

## Not In Scope (Current)

- Cloud sync
- Real-time collaboration
- Multi-agent orchestration
