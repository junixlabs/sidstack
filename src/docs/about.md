# About SidStack

**SidStack** is a knowledge-powered AI coding assistant platform designed for developers who work with AI coding tools like Claude Code.

## What is SidStack?

SidStack helps you:

- **Understand** your project through structured knowledge and impact analysis
- **Track** tasks and AI agent progress with governance and quality gates
- **Organize** your codebase with structured knowledge management
- **Analyze** change impact before implementing

## Core Principles

### Knowledge-First
Understand your project before automating. SidStack provides tools to browse and organize technical documentation.

### Visual & Friendly
Both humans and AI agents can read and understand the same information. No hidden configurations.

### Local-First
Everything runs on your machine. No cloud dependencies, no external servers required.

### Incremental
Start small and grow. Add knowledge incrementally as your project evolves.

## Key Features

| Feature | Description |
|---------|-------------|
| **Project Hub** | Capability tree and entity connections |
| **Knowledge Browser** | Browse and search project documentation |
| **Task Manager** | Track AI work with governance and quality gates |
| **Ticket Queue** | External ticket intake, review, convert to tasks |
| **Training Room** | Capture lessons from incidents, build rules |

## Technology Stack

- **Desktop App**: Tauri 2.x (Rust + React)
- **MCP Server**: Claude Code integration (20 tools)
- **CLI**: Oclif v3 (TypeScript)
- **Database**: SQLite (better-sqlite3, embedded)
- **Styling**: Tailwind CSS

## Getting Started

1. Open a project folder with SidStack
2. Run `sidstack init --scan` to initialize and generate knowledge docs
3. Browse your project in the Project Hub
4. Search documentation in the Knowledge Browser
5. Track work in the Task Manager
